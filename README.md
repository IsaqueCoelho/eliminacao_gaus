# Gauss-Jordan Elimination Website
Construção de um WebSite que calcule matrizes através do método de Gauss, considerando matrizes com tamanhos de 2 a 6!
  
<br/>
<br/>
  
## Tecnologias

### Front-end
* HTML5.
* CSS.
* Javascript.

### Back-end
PHP.
